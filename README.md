# Dotfiles

Development machine provisioning via AI agent. See [`AGENTS.md`](AGENTS.md) for setup instructions.

Uses [Devbox](https://www.jetify.com/devbox) for package management and [GNU Stow](https://www.gnu.org/software/stow/) for config symlinking.
